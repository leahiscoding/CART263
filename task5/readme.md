1) Emile's Interactive Bike Garage

 

I found this website well-designed. It was the one that stood out to me the most. I really enjoyed the animation that appears when you hover over the bike, showing two different features of the website. It's also impressive to see the individual animation for each function when I hover my mouse over them. The UI is very minimalistic and pleasing. I liked how Émile created the shape of a bike using lines and made it geometric.

 

Although I live in Montreal and rarely bike or use BIXI, I found the information on bike maintenance to be helpful. The details were concise and clear—easy to read and understand, even without prior knowledge of bikes. I also liked the feature that lets you add either a part or a bike, depending on your needs. You can also add colours, which adds some diversity to your bikes. When I added a bike, it displayed the anatomy of my bike and input boxes for the toolbox. I find this tool extremely useful.

 

I would love to see this implemented in real life, possibly showcased officially or locally, and hosted within an app. One feature I would like to add is the ability to delete individual bikes instead of deleting all of them. This website motivated me to try oiling some bikes in my parents' garage that I haven't touched in a while.

 

 

2) Nou Noune's class notes

 

This was an interesting approach I hadn’t considered before. When I first received this assignment, I thought it had to be something unrelated to the class notes. I assumed the project needed to be approached more artistically. This project surprised me because I never thought of turning a class note into a project. It was an original and creative way to review the material.

 

I really appreciated how Nou Noune integrated each section of the project into a fun and engaging playground. The visual elements are simple yet amusing. The vibrant, joyful colours and shapes give a glimpse of Nou Noune’s artistic personality, alongside the pixelated character.

 

My favourite part of the project was the grid, as it was visually pleasing. I liked how the squares are filled with pastel pink in a seemingly random pattern. It felt both random and orderly at the same time. I also loved the character within objects—they were so adorable. Overall, this was a great opportunity for me to revise the class material.

 

3) Jordan Labasso’s project 1

 

Honestly, this website felt simple yet artistic. I was curious about how it worked, so I inspected the code to see how it was written. Despite the code being relatively short, I had difficulty understanding its structure.

 

The website is straightforward and charming. It features three sliders with a black box. The sliders are labelled: matrix, gravity, and extend. I’m not entirely sure how they influence each other, but adjusting them shows noticeable differences.

 

As you hover the cursor over the black screen, sand appears to fall from the tip of the cursor, like pouring from a bucket. I think it was a good choice to make the sand in pixel art style, as it creates an interesting visual. I also appreciate how the sand changes colour along the rainbow spectrum as time passes. Overall, it was a fascinating project to interact with.
