# an einem seltsamen Ort

Final project for CART 263.

## Project Link

https://leahiscoding.github.io/CART263/finalProject

## Description

an einem seltsamen Ort means In a Strange Place.

This project is an interactive website inspired by German techno music, night culture, and the feeling of entering an unfamiliar digital space. The website includes different interaction sections such as Face-Mesh, Handpose, and Speech Recognition. These sections allow the user to interact with the website through the face, hand, and voice.

The goal of the project is to create a strange, playful, and music-based web experience rather than a normal static webpage.

## Project Process

This project started while I was listening to German techno music during a run. I wanted to create something that responded to the mood and rhythm of the music. I first designed the website in Figma, then built it with HTML, CSS, and JavaScript.

I originally researched PyTorch, TensorFlow.js, and MediaPipe, but I decided to use ml5.js because it worked better with p5.js and was more practical for a browser-based creative coding project.

## Use of Generative AI

Generative AI tools such as Codex and Claude were used to support the development process. They helped with troubleshooting, CSS adjustments, Figma MCP workflow, and understanding code structure. However, I reviewed, tested, and edited the code myself.
